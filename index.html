Cara Menggunakan Media

Selamat datang di Media Interaktif Disaster Report! Ikuti petunjuk berikut untuk pengalaman belajar terbaik:

  • Eksplorasi Materi: Gunakan menu Learn dan Example untuk memahami struktur Report Text.
  • Simulasi: Klik menu Simulation untuk melihat bagaimana bencana alam terjadi secara visual.
  • Sistem Poin: Selesaikan tantangan di menu Games untuk mengumpulkan poin:
    • Level 1: Menjodohkan kosakata (10 poin/item).
    • Level 2: Benar/Salah (20 poin/soal).
    • Level 3: Kuis Pilihan Ganda (30 poin/soal).
  • Sertifikat: Setelah menyelesaikan semua level, Anda dapat mencetak sertifikat di menu Certificate.
  • Audio: Klik ikon speaker di pojok kiri bawah untuk menyalakan/mematikan suara.

Tujuan Pembelajaran

Tujuan Pembelajaran

  1. Menganalisis fungsi sosial, struktur, dan unsur kebahasaan Report Text terkait fenomena alam.
  2. Menemukan informasi rinci, makna tersurat, dan makna tersirat mengenai Bencana Alam.
  3. Mengaplikasikan literasi mitigasi bencana ke dalam strategi keselamatan diri.

What is Report Text?

Definition

Report text is a text that presents information about something as it is. It is a result of systematic observation and analysis. We focus on Natural Disasters.

Generic Structure

  • General Classification: Menjelaskan fenomena secara umum.
  • Description: Mendeskripsikan detail karakteristik, penyebab, atau dampak fenomena.

Language Features

  • Simple Present Tense: Menyatakan fakta ilmiah.
  • General Nouns: Membahas objek secara universal.
  • Action Verbs: Kata kerja tindakan.
  • Technical Terms: Istilah ilmiah terkait bencana.
